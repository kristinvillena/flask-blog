# Flask Blog Web Application

A blog-style web application built using **Flask (Python)**.  
Users can **register**, **log in**, **create posts**, **edit posts**, and **log out**.

This project was built to learn full-stack web development concepts including:
- Routing and URL handling
- HTML templating with Jinja2
- User authentication and session management
- Database CRUD operations (SQLite)
- Project structure using the application factory pattern

---

## Features

| Feature | Description |
|---------|-------------|
| User Registration | Create an account with username & password |
| Login / Logout | Secure session authentication |
| Create Post | Authenticated users can write blog posts |
| Edit / Delete Post | User can only edit/delete their own posts |
| Templates | Dynamic content rendered via **Jinja2 templates** |
| SQLite Database | Simple persistent storage |

---

## Tech Stack

| Layer | Technology |
|-------|------------|
| Backend Framework | Flask (Python) |
| Frontend | HTML, CSS, Jinja2 Templates |
| Database | SQLite |
| Testing | Pytest |
| Deployment Ready For | Render / Azure App Service |
